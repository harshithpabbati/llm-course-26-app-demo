## Problem

Desktops can get cluttered very easily, especially for busy people. As such, it can get difficult to search for the correct files, or even delete files that are no longer needed. Sorting can take a very long time, as well.

The question, therefore, is: **how can we sort an incredibly cluttered desktop without spending too much time?**


## Solution

This desktop sorter. It is an application that searches through all the files and applications in the desktop and sorts them into a folder. Any files that can be up for deletion (files last opened after a certain number of days) are also put into folders in order to do a double check as to whether they should be deleted or not


## User Flow

- View the project page like any visitor would
- Notice the embedded Google Drive video at the top
- Scroll through the standard sections to see the layout


## LLM Components

- **Folder Sorting** - The application uses LLAMA 3.2 to determine whether a file fits a certain category. Files that do not fit any of the categories will be sorted by their file type.
